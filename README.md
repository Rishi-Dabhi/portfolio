# Portfolio - Rishi Dabhi

Personal portfolio site built with Next.js, Framer Motion, and Tailwind CSS.

## Tech Stack

- **Next.js 16** (App Router, Turbopack)
- **React 19**
- **Framer Motion** — scroll animations, 3D tilt cards, parallax effects
- **TypeScript**
- **Tailwind CSS v4** + custom CSS design tokens


## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
